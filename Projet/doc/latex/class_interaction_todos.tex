\doxysubsection{Detailed Description}
Container for \mbox{\hyperlink{class_interaction_todo}{Interaction\+Todo}} objects. 

A container class for \mbox{\hyperlink{class_interaction_todos}{Interaction\+Todos}}. Represents the idea that a Relational Database only has {\bfseries{atomic values}}~\newline
 It is not a list of pointers because an \mbox{\hyperlink{class_interaction_todo}{Interaction\+Todo}} has no reason to exist out of this list. Therefore every time there is the need to use an element of this list, a pointer towards it has to be created.\hypertarget{class_todo_ex}{}\doxysubsubsection{Example}\label{class_todo_ex}
