\doxysubsection{Detailed Description}
Simplifies the use of C++ tm struct to represent dates. Uses the dd/mm/yyyy format. 

Custom interface built from the tm struct~\newline
 It allows easy management of dates using only three parameters, day, month and year.~\newline
 Therefore, the class is easily adjustable so we can later include hours, minutes, seconds if necessary.
