\doxysubsection{Detailed Description}
Container for \mbox{\hyperlink{class_log}{Log}} objects. 

\mbox{\hyperlink{class_logs}{Logs}} is a container for \mbox{\hyperlink{class_log}{Log}} objects. ~\newline
 The point of this class is nothing more than to make manipulating lists of logs more friendlier using add, get, etc... methods. Consider it equivalent to std\+::list$<$\+Log$>$
