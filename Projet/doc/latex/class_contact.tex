\doxysubsection{Detailed Description}
A contact object represents an entity any user can interact with. 

A contact represents an entity akin to a person, a group of persons, a service, a company...~\newline
 It is the main element of the class -\/ Or at least the one that will be iterated onto the most along Interactions.~\newline
 A contact stores all of its basic informations (names, mail, etc...), a list of interactions.~\newline
